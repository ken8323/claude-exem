# TODOアプリケーション

シンプルで使いやすいTODO管理アプリケーションです。HTML、CSS、JavaScriptのみで構築されており、ブラウザで直接動作します。

## 機能

### 基本機能
- **タスクの追加**: 新しいTODOを簡単に追加
- **タスクの編集**: 既存のTODOを編集
- **タスクの削除**: 不要なTODOを削除
- **完了管理**: チェックボックスで完了/未完了を切り替え

### 高度な機能
- **優先度設定**: 高・中・低の3段階で優先度を設定
- **期限管理**: タスクに期限を設定し、期限切れを視覚的に表示
- **カテゴリー分類**: タスクをカテゴリーで分類
- **フィルタリング**: カテゴリーや状態でタスクを絞り込み
- **ソート機能**: 作成日順、優先度順、期限順で並び替え
- **データ永続化**: LocalStorageを使用してデータを保存

## 使い方

### 起動方法
1. `index.html`をブラウザで開く
2. すぐに使い始められます

### タスクの追加
1. 「新しいTODOを追加」フォームにタスク情報を入力
   - タイトル（必須）
   - 説明（オプション）
   - カテゴリー（例: 仕事、個人）
   - 優先度（高・中・低）
   - 期限日
2. 「追加」ボタンをクリック

### タスクの編集
1. 編集したいタスクの「編集」ボタンをクリック
2. フォームに現在の値が入力される
3. 必要な変更を行い「更新」ボタンをクリック

### タスクの削除
1. 削除したいタスクの「削除」ボタンをクリック
2. 確認ダイアログで「OK」をクリック

### フィルタリングとソート
- **カテゴリーで絞込**: ドロップダウンから特定のカテゴリーを選択
- **状態で絞込**: すべて/未完了/完了済みを選択
- **並び替え**: 作成日順/優先度順/期限順を選択

## 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: レスポンシブデザイン、Flexbox、Grid
- **JavaScript (ES6+)**: クラスベースのオブジェクト指向設計
- **LocalStorage**: クライアントサイドのデータ永続化

## カスタマイズ

### 色のカスタマイズ
`styles.css`の`:root`セクションでカラー変数を変更できます:

```css
:root {
    --primary-color: #4a90e2;
    --priority-high: #e74c3c;
    --priority-medium: #f39c12;
    --priority-low: #27ae60;
}
```

### 機能の拡張
`app.js`のクラスを拡張することで、新しい機能を追加できます。
