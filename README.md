# Claude Code Examples

このリポジトリはClaude Codeで作成したサンプルプロジェクトのコレクションです。

## プロジェクト一覧

### [TODO管理アプリ](todo/)

シンプルで使いやすいTODO管理アプリケーション

**技術スタック:**
- HTML5
- CSS3 (レスポンシブデザイン)
- JavaScript (ES6+)

**主な機能:**
- タスクの追加・編集・削除
- 優先度設定（高・中・低）
- 期限管理
- カテゴリー分類
- フィルタリング＆ソート
- LocalStorageでのデータ永続化

**特徴:**
- 外部ライブラリ不要
- レスポンシブ対応（PC・タブレット・スマホ）
- 2カラムレイアウト（大画面）

詳細は[todo/README.md](todo/README.md)を参照してください。
